# [LIVE DEMO](https://elegant-pastelito-17a728.netlify.app/)

This is a solo project from the javascript course of the coding school Scrimba. The assignment was to build a functional basketball scoreboard given html/css skeletons and a figma design file.

## Getting Started

Install the dependencies and run the project

```
npm install
npm start
```

Head over to https://vitejs.dev/ to learn more about configuring vite
